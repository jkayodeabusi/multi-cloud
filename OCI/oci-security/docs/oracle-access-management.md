# Oracle Access Management (OAM)

Oracle Access Management provides Single Sign-On (SSO), Multi-Factor Authentication (MFA), federation, and application integration within OCI Identity Domains.

---

## Accessing OAM via OCI Console

1. Open the [OCI Console](https://cloud.oracle.com)
2. Sign in with:
   - Cloud account name (tenancy)
   - Username
   - Password / MFA
3. Open the navigation menu (☰)
4. Navigate to: **Identity & Security → Identity → Domains**
5. Select your identity domain (usually `Default`, or a custom domain)

---

## What You Can Manage

| Feature | Description |
|---------|-------------|
| **Users** | Create, update, deactivate user accounts |
| **Groups** | Organize users into logical groups |
| **MFA** | Configure multi-factor authentication policies |
| **SSO** | Set up Single Sign-On for applications |
| **Applications** | Register and manage connected apps |
| **Federation** | Integrate with external IdPs (Entra ID, Okta, etc.) |
| **Policies** | Define access and authentication policies |
| **API Authentication** | Manage API keys and OAuth clients |
